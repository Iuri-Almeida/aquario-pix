# Aquario-Pix

## Description

Microservices-based backend application to manage a pix transaction. Please see the project [statement][statement] (pt-BR) for details.

## How-To

### Requirements

- Docker Compose

### Running

```shell
    $ git clone https://github.com/Iuri-Almeida/aquario-pix
    
    $ cd aquario-pix
    
    $ docker-compose up
```
